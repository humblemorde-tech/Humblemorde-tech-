## Pair: `https://your-app.com`
Enter number > Get 8-digit code > Link in WhatsApp

## Commands: 40 Total
`.menu` shows all. Categories: general 10, group 10, downloader 8, tools 8, owner 4

## Env Variables
| Variable | Description |
| --- | --- |
| `OWNER_NUMBER` | Your number. Ex: `254712345678` |
| `PREFIX` | Command prefix. Default `.` |
| `SESSION_NAME` | Session folder. Default `session` |

## Warning
